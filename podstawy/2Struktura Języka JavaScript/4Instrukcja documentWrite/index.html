Instrukcja document.write

Instrukcja document.write() - zapisuje ciąg tekstowy (używana do testowania).

document - obiekt JS, który reprezentuje aktualnie wyświetlaną stronę

write() - METODA, czyli FUNKCJA WYKONUJĄCA OKREŚLONE DZIAŁANIE NA OBIEKCIE.

TEKST UMIESZCZONY W NAWIASACH - ARGUMENT wywołania metody.

Ogólny zapis:

obiekt.metoda(argumenty metody);

ZNACZNIKI HTML W document.write()

Wewnątrz instrukcji można używać znaczników HTML:

  • Przyklad 1:
    • document.write("

      JavaScript

      ");
  • Przyklad 2:
    • document.write("Spis treści");
  • Przyklad 3:
    • document.write(201);
  • Przyklad 4:
