Wprowadzenie - struktura języka JavaScript

Po utworzeniu kodu HTML i arkusza CSS należy napisać kod kontrolujący zachowanie strony oraz interakcje z użytkownikiem.

    Do zadań JAVASCRIPT należą:
  • Prosta manipulacja danymmi
  • Dynamiczne modelowanie struktury strony
  • Obsługa rozszerzeń multimedialnych
  • Tworzenie odrębrnych aplikacji

Język JAVASCRIPT pozwala na:

  • Tworzenie i umieszczanie w kodzie krótkich programów, które:
    • obsługują zdarzenia
    • weryfikują dane wprowadzane do formularza
    • nawigują między stronami
  • Użycie bibliotek rozszerzających możliwości (np. jQuery)
