Opis Języka JAVASCRIPT

Język JAVASCRIPT - język OBIEKTOWY SKRYPTOWY nawiązującym do języka C.

Jest to język INTERPRETOWANY - nie wymaga kompilowania kodu.

Potrzebna jest tylko przeglądarka obsługująca JS.

Typowe zastosowania JS:

  • modyfikowanie wyglądu dokumentu
  • wyświetlanie okien dialogowych
  • kontrola poprawności wypełniania formularza
  • manipulacja info o dacie i czasie
  • lokalne generowanie dokumentów HTML
