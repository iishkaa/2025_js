Instrukcje w JS:

Język JS ma określone słowa kluczowe:

  • wyrażenia
  • słowa kluczowe
  • komentarze
  • zmienne
  • literały
  • operatory
  • zbiór predefiniowanych obiektów i funkcji

PRZYKLAD:

  • var a, b, c; - deklaracja zmiennych
  • a = 4; b = 9; - przpisanie wartości
  • c = a + b; - wyrażenie

INSTRUKCJE - mogą być pisane:

  1. pojedynczo
  2. w jednym wierszu
  3. kilku wierszach

ŚREDNIK - rola ogranicznika instrukcji (nie jest wymagany jeśli po instrukcji wystąpi znak nowego wiersza.

UWAGA: Jeśli kilka instrukcji zostanie zapisane w JEDNYM wierszu MUSZĄ być oddzielone ŚREDNIKIEM.

JS ignoruje powtarzające się SPACJE

OPERATORY MATEMATYCZNE: + - / * %

OPERATORY LOGICZNE: && || !

Instrukcje mogą być GRUPOWANE W BLOKI za pomocą nawiasów {}
